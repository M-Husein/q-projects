Frequently Asked Questions

PT. Bank Rakyat Indonesia (Persero) Tbk.

1. Apakah Bri Quote?

BRI Quote adalah layanan informasi terintegrasi untuk memperoleh informasi counter rate dan deal rate bank notes di BRI

2. Bagaimana cara log ini di BRI Quote?

  1. Input user name
  2. Input password
  3. Klik Authorized Money Changer (untuk Pedagang Valuta Asing)
  4. Klik Log In

3. Bagaimana mengetahui informasi counter rate?

  1. Pilih Home
  2. Pilih Information
  3. Pilih Counter Rate

4. Bagaimana memperoleh informasi money changer outlet?

  1. Pilih Home
  2. Pilih Information
  3. Pilih Money Changer Outlet
  4. Klik Nearest Branch
  5. Klik Branch Marker yang muncul
  6. Klik Call untuk menghubungi Kanca

5. Bagaimana memperoleh data stok bank notes dan nego?

  1. Pilih Home
  2. Pilih Quotation Services
  3. Input Area
  4. Input Currency
  5. Klik Check BN Availibility

6. Bagaimana memperoleh informasi kurs nego?

  1. Pilih Home
  2. Pilih Quotation Services
  3. Input Area
  4. Input Currency
  5. Klik Check BN Availibility
  6. Klik Continue
  7. Input Branch
  8. Input From Currency
  9. Input To Currency
  10. Input Denom
  11. Isi Quantity
  12. Klik Add jika ada tambahan denom
  13. Browse pic jika total > USD 25.000
  14. Klik Submit
  15. Muncul Form Konfirmasi Booking
  16. Klik Download (jika ingin di svave di device)

7. Bagaimana cara menghitung kurs secara online?

  1. Pilih Home
  2. Pilih Converter Rate
  3. Pilih Kurs
  4. Input Nominal
  5. Klik Calculate
  6. Muncul Result

8. Bagaimana jika ingin mengajukan saran?

Silahkan ajukan saran anda ke Divisi Bisnis Internasional Bagian Bisnis Money Changer Telp. (021) 575-2782/84
